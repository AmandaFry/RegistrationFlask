  • email

  • password
  • confirm_password

Validate must include:

  • All fields are required and must not be blank.
  • First and Last Name cannot contain any numbers.
  • Password should be more than 8 characters. Optional advenced option: password to have at least 1 uppercase letter and 1 numeric value
  • Email should be a valid email.
  • Password and Password Confirmation should match
  • Optional advenced option: Add a birth-date field that must be validated as a valid date (and must be from the past).
